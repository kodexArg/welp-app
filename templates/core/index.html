Roles del Sistema

Conocer en detalle el flujo de Payflow →
Usuario Final

Operativos, administrativos y todos los empleados de la empresa.

Técnico

Responsables del área de sistemas. Presupuestan y resuelven incidencias.

Supervisor

Líder, supervisor o jefe de área. Realiza la primera autorización de solicitudes.

Responsable de Compras

Rol específico encargado del sistema de pagos y de la gestión de presupuestos con proveedores.

Manager

Gerente de UDN. Realiza la primera firma obligatoria para autorización de pagos.

Director ⭐

Nivel ejecutivo. Realiza la segunda firma obligatoria para autorización de pagos. Control total del sistema.

Los permisos se asignan por UDN y Sector para simplificar la gestión.

Tecnología y Arquitectura

Stack Principal

  • Backend: Python 3.11, Django 5, Gunicorn
  • Base de Datos: PostgreSQL en AWS RDS
  • Almacenamiento de Archivos: AWS S3 con CDN CloudFront
  • Despliegue Continuo: AWS App Runner
  • Frontend Moderno: Vite, Tailwind CSS v4, HTMX y Django Components
